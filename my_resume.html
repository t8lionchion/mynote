我是蔡德誠，畢業於國立彰化師範大學數學系資訊組。四年的大學學習不僅培養了我扎實的邏輯思維與數學基礎，也讓我對程式開發產生了濃厚的興趣。在學期間，我接觸到 Python、Kotlin 與 MySQL 等技術，並完成了數個具有挑戰性的專案，逐步奠定了我對後端開發的熱情與專業基礎。 在學期間，我曾完成火車查詢系統的專案，負責後端架構設計、資料庫建模與 API 開發。該專案使用 Python 編寫商業邏輯，並採用 MySQL 儲存並管理火車數據，成功實現即時的時刻表與票價查詢功能。在這個過程中，我學會如何從需求分析到系統設計進行完整的實作，也逐漸熟悉如何將前後端整合，提升使用者體驗。 除了專案經驗，我對學習新技術充滿熱情。從接觸程式設計開始，我經常參與線上課程與技術社群，主動探索如何運用程式解決實際問題。例如，使用 Python 連接台鐵 API 獲取資料，並將整理後的資料匯入資料庫。依據設計的三項火車查詢功能，分別開發三個伺服器（Server），最後以 Kotlin 開發簡易的手機應用程式並串接 API。我相信持續學習是技術人的核心競爭力，而這也正是我樂於挑戰未知的原因。 在性格上，我是一個責任感強、重視細節的人，對於需要邏輯推演與精準實作的任務充滿興趣。我過去曾參與多次團隊合作，深知溝通與協作的重要性，也能快速適應並融入新的工作環境。我相信這些特質，將使我成為一名值得信賴的後端開發工程師。 展望未來，我希望能在一個以技術為核心的團隊中發揮專長，積極參與後端架構設計、資料庫優化與 API 開發，並持續學習雲端技術與微服務架構。我期待藉由實際參與開發專案，不斷提升自己的技術能力，為公司創造價值的同時，逐步實現成為資深工程師的職涯目標 。
